# Élégance - Salon de Coiffure

Site vitrine moderne pour un salon de coiffure premium situé à Paris.

## Description

Ce site présente les services, produits et réalisations du salon de coiffure "Élégance". Il permet aux clients de prendre rendez-vous, de découvrir la galerie de réalisations et de consulter les prestations proposées.

## Fonctionnalités

- Présentation des services pour femmes, hommes et enfants
- Galerie photo avec filtres et lightbox
- Formulaire de prise de rendez-vous
- Informations de contact et localisation
- Design responsive et moderne
- Animations fluides avec Framer Motion

## Technologies utilisées

- React 18
- Tailwind CSS
- Framer Motion
- Lucide React (icônes)
- React Router
